# ☕ VUT Café Website

A modern and responsive website designed to improve the campus dining experience at the **Vaal University of Technology**.

## 🚀 Features

- 🍔 View the café menu
- 🕒 Check operating hours
- 📍 Find the café location
- 📱 Responsive design
- ⚡ Simple and easy navigation

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📖 About

The **VUT Café Website** provides students and staff with quick access to café information through a clean, user-friendly interface. It makes it easier to browse the menu, view opening hours, and stay informed about the campus café.

## 🎯 Future Improvements

- Online ordering
- Student login
- Online payments
- Daily specials
